# About the Creator

## Rohan Sharma

Radhika's DocManager was built by **Rohan Sharma**, a passionate developer who believes in building tools that are both powerful and easy to use.

## The story behind the name

Radhika's DocManager is named after **Radhika Sharma**, a special person whose memory inspired this project. The R in R's DocManager stands for Radhika, and the application was built as a tribute to her lasting impact. (it's an imaginary memory)

> Built by Rohan Sharma, driven by the memories of Radhika Sharma.

## What makes this project special

- Security-first design with AES-256-GCM encryption and bcrypt hashing
- User-centric interface with a clean, minimal design
- AI without vendor lock-in using your own API keys
- Multi-organization architecture ready for enterprise use
- Four tiers of carefully designed role-based permissions
