# Getting Started

Get up and running with R's DocManager in minutes.

## Creating Your Account

1. Click "Get Started" on the landing page
2. Fill in your details: full name, email, password, and organization
3. Click "Register" and you will be logged in automatically

## Logging In

1. Go to the Sign In page
2. Enter your email and password
3. You will be redirected to your dashboard

## Your Dashboard

After logging in, you will see:

- Recent documents for quick access
- Document count and organization info
- Quick actions to create or browse documents

## Navigation

| Link | Description |
|------|-------------|
| Dashboard | Your personal overview |
| Documents | Browse and search all documents with tabs |
| Users | Manage organization users (admin and above) |
| God Panel | Platform-wide control (god only) |
| Docs | This documentation |
| New Doc | Create a new document |

## Setting Up AI

1. Click your profile icon then AI API Keys
2. Select a provider: Groq (free tier), OpenAI, or Anthropic
3. Paste your API key and save
4. Keys are encrypted with AES-256-GCM before storage

Note: Local tools (Word Count, Structure Analysis, Preview) work without any API key.

## Your First Document

1. Click "New Doc" in the navigation bar
2. Choose Upload (drag and drop a file) or Write (Markdown editor)
3. Add a title (required) and optional description, tags
4. Set the status: Draft, Under Review, Published, or Archived
5. Set the classification: Organization, Public, Internal, Confidential, or General
6. Set the access level: View Only, Comment, Edit, or Full Access
7. Optionally assign reviewers (for Under Review status) and reference other documents
8. Click "Create Document"

## Understanding Document Tabs

After creating documents, you can find them in different tabs:

- Published: documents with status Published
- Drafts: your own draft documents
- Review: documents with status Under Review
- Archived: your own archived documents
- For Me: documents where you are assigned as a reviewer
