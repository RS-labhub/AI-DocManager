# Welcome to Radhika's DocManager

![banner](/og.png)

Radhika's DocManager (R's DocManager) is a secure, AI-powered document management platform built for teams and organizations. Upload, organize, and enhance your documents with built-in AI features while keeping your data safe with enterprise-grade security.

## What can you do?

- **Upload documents** -- PDF, DOCX, TXT, CSV, Markdown, HTML, JSON, XLSX, PPTX, RTF, ODT with automatic text extraction for supported formats
- **Write rich content** -- Built-in Markdown editor with live preview and RAG file import
- **Document viewers** -- PDF iframe viewer, Word document viewer (extracted text), Markdown formatted preview, plain text display
- **AI-powered actions** -- Summarize, analyze, extract keywords, improve writing, translate, Q&A, and generate insights
- **Organize with status** -- Draft, Under Review, Published, Archived with tab-based filtering
- **Classify documents** -- Organization, Public, Internal, Confidential, or General classifications
- **Control access levels** -- View Only (no comments), Comment, Edit, or Full Access
- **Password-protect documents** -- Lock sensitive documents with a 9-digit code
- **Assign reviewers** -- Select admins and super admins to review documents
- **Reference documents** -- Link related documents together
- **Multi-organization support** -- Complete data isolation between organizations
- **Role-based access** -- Four roles (User, Admin, Super Admin, God) with granular permissions
- **Bring your own AI keys** -- Use Groq, OpenAI, or Anthropic with AES-256-GCM encrypted key storage
- **Free local tools** -- Word count, structure analysis, and text preview without API keys

## Quick Start

1. Create an account at the registration page
2. Join or create an organization during signup
3. Upload your first document using the New Doc button
4. Set classification, access level, and status
5. Try AI actions on any document with content
6. Invite team members if you are an admin

## Who is this for?

- Teams that need a central place to manage and collaborate on documents
- Organizations that require strict access control and data isolation
- Developers who want AI-enhanced document workflows without vendor lock-in
